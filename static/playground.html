CONTROLS

• DRAG a library item onto the canvas to add (or click)
• DRAG nodes to reposition
• DRAG bottom slot → top slot to delegate
• DRAG skill side slot → agent side slot for tools
• DOUBLE-CLICK an edge to delete it
• SHIFT+CLICK adds a node to the selection
• SHIFT+DRAG empty canvas to lasso-select
• DRAG canvas to pan · SCROLL to zoom · F to fit
• DEL removes everything selected
